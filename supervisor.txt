this is the api contract with rules
username*	string
example: customUserName1234
minLength: 5
maxLength: 20
Unique username for the actor

password*	string
example: 12345678
minLength: 8
maxLength: 20
Password for authentication

email	string
example: user@gmail.com
Email address of the actor

phone	string
example: 0953464571
Phone number in Syrian format

accountStatus	string
Current status of the actor account

Enum:
[ ACTIVE, NOT_ACTIVE, COMPLETE_REGISTRATION_REQUIRED, BLOCKED, DELETED ]
roleId*	string
example: 123e4567-e89b-12d3-a456-426614174000
UUID of the role to assign

profile*	CreateProfileDto{
firstName*	string
example: John
maxLength: 50
lastName*	string
example: Doe
maxLength: 50
midName	string
example: Michael
maxLength: 50
additionalName	string
example: Jr.
maxLength: 50
birthDate	string($date-time)
example: 1990-05-15T00:00:00.000Z
birthPlace	string
example: Damascus
maxLength: 100
nationalId	string
example: 1234567890
maxLength: 20
imageId	string
example: 1234567890
should be correct uuid of existing image

job	string
example: Engineer
maxLength: 100
education	string
example: Bachelor's Degree
maxLength: 100
address	string
example: Damascus, Syria
maxLength: 255
distinguishingSigns	string
example: Has a scar on left hand
maxLength: 255
note	string
example: Important note about this person
maxLength: 255
}
}
